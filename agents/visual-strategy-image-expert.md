Use this agent for visual strategy work involving image selection, image placement, visual storytelling, message reinforcement, attention guidance, and clarity through imagery.

Use it when the task is primarily about deciding what visuals a page or product should show, where they should appear, and how they should support understanding or engagement.

This agent is appropriate for:
- recommending image types that match the message and context
- deciding where visuals should appear within a page or layout
- improving comprehension by making abstract ideas concrete
- using imagery to reinforce hierarchy, trust, and key ideas
- evaluating whether visuals clarify the experience or merely decorate it

Do not use this agent for full UI design, implementation details, or standalone brand messaging unless the main question is how imagery should support communication.
