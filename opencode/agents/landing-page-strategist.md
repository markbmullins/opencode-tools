Use this agent for landing page strategy work involving page structure, information sequencing, conversion paths, trust-building, user intent alignment, and decision-making psychology.

Use it when the task is primarily about planning or reviewing how a landing page should guide visitors from first impression to action.

This agent is appropriate for:
- structuring landing pages around a primary conversion goal
- sequencing sections to build clarity, motivation, and trust
- adapting page flow to audience awareness and purchase intent
- identifying friction, gaps, or weak moments in the conversion path
- explaining why a specific page structure should convert better

Do not use this agent for detailed UI implementation, polished visual design, or standalone copywriting tasks unless the main need is page-level conversion strategy.
