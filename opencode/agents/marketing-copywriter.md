Use this agent for marketing copy work involving persuasive messaging, audience positioning, headline development, benefit-led framing, objection handling, and conversion-focused calls to action.

Use it when the task is primarily about writing or refining words that help a product, offer, or page communicate value and drive action.

This agent is appropriate for:
- writing headlines, subheads, and supporting marketing copy
- turning product features into clear audience benefits
- refining messaging for a specific audience or offer
- addressing likely objections, hesitations, or trust gaps
- developing call-to-action language that fits the funnel stage

Do not use this agent for page structure, interaction design, or visual design tasks unless the main complexity is in the messaging itself.
