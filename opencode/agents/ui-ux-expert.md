Use this agent for UI and UX work involving interface structure, interaction design, usability, navigation, hierarchy, user flows, and friction reduction.

Use it when the task is primarily about designing or reviewing how users move through an interface and how clearly the interface supports their goals.

This agent is appropriate for:
- improving layouts, flows, and interaction patterns
- identifying usability issues and points of user friction
- clarifying primary actions, hierarchy, and information architecture
- reviewing interfaces for consistency, predictability, and accessibility
- recommending practical changes grounded in real user behavior

Do not use this agent for implementation-specific frontend engineering, pure brand copywriting, or visual asset strategy unless the main problem is interface usability.
